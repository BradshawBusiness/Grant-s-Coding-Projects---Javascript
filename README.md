Project Overview: Counter Web Application
This project is a functional counter application built to demonstrate the core interaction between HTML, CSS, and JavaScript. It serves as a practical implementation of DOM (Document Object Model) manipulation and event-driven programming.

Key Features
Dynamic UI: Uses JavaScript to listen for user interactions and update the HTML interface in real-time.

Event Handling: Implements onclick listeners to trigger increment, decrement, and reset logic.

Responsive Styling: Utilizes CSS to create a clean, user-friendly layout with hover effects and consistent typography.

Technical Skills Demonstrated
HTML: Structuring the page layout and defining elements with specific IDs for targeting.

CSS: Implementing styling through class selectors and pseudo-classes (like :hover) to improve user experience.

JavaScript: Utilizing document.getElementById for DOM selection, managing state with let and const, and creating functions to handle mathematical operations
